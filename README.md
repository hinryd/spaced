## Spaced

Basically a mimicking the `gap` property in a CSS `flexbox`

## Features

- Provide `SpacedColumn` and `SpacedRow` widgets

## Getting started

`flutter pub add spaced`

## Usage

1. Add `Spaced` in front of your `Column` or `Row` widget
2. Add `spacing` to property
3. Profit!
